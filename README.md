# Python OOP Basics

This repository contains a beginner-level Jupyter Notebook used to practice
basic Object-Oriented Programming (OOP) concepts in Python.

## Concepts Covered
- Class definition
- Constructors (`__init__`)
- Instance attributes
- Basic object creation

## Purpose
This project is for learning and practicing Python OOP fundamentals.
The notebook is intentionally simple and focused on core concepts.
